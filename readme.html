Execution

The submission is a WebStorm project that can be run directly from the IDE.

Assumptions

The site use bootstrap CDN therefore assumes an internet connection is available.

The list of locations cant have same locations name or the same location coordination. in that case the user will get error message

The input validation happens in the html with html 5 validation.

In the html the inputs use the form method with submit. after the submit button trigger java script catch the submit and handle the inputs with own way.

Each inputs coordinate are saved in Database arr. when the show location button trigger, the fetch use the relevant location from the arr location

The Location list is implement with drop down list

Error massage shown to the user with model dialog

The weather for 7 days will shown in bootstrap table

The 7 days weather information will fetch from the server , the weather picture of the weather will get from url without fetch method
